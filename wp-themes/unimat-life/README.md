# UNIMAT LIFE WordPress Theme

UNIMAT LIFEコーポレートサイト用WordPressテーマ

## インストール方法

1. このテーマフォルダを `wp-content/themes/` ディレクトリにアップロード
2. WordPress管理画面の「外観」→「テーマ」から「UNIMAT LIFE」を有効化

## 必要な設定

### メニューの設定

1. WordPress管理画面の「外観」→「メニュー」にアクセス
2. 新しいメニューを作成
3. メニュー項目を追加（サブメニューも設定可能）
4. 「メニューの位置」で「Primary Menu」を選択して保存

### ロゴの設定

1. WordPress管理画面の「外観」→「カスタマイズ」→「サイト識別情報」にアクセス
2. 「ロゴ」をアップロード
3. 推奨サイズ: 高さ44px（幅は自動調整）

### 画像ファイル

以下の画像ファイルを `assets/images/` ディレクトリに配置してください：
- `logo.png` - シンボルロゴ
- `logo_type.png` - タイプロゴ

## テンプレート構造

- `front-page.php` - トップページテンプレート
- `index.php` - メインテンプレート（投稿一覧・個別投稿）
- `header.php` - ヘッダー部分
- `footer.php` - フッター部分
- `functions.php` - テーマ機能
- `style.css` - スタイルシート

## 機能

- レスポンシブデザイン（PC・スマートフォン対応）
- メガメニュー（PC表示時）
- ハンバーガーメニュー（スマートフォン表示時）
- 検索機能
- カスタムロゴ対応
- アイキャッチ画像対応

## カスタマイズ

### ニュース表示

トップページのニュースセクションは、WordPressの投稿機能を使用しています。
最新の投稿が3件表示されます。

### サービス紹介

サービス紹介セクションは `front-page.php` で直接記述されています。
必要に応じてカスタム投稿タイプやカスタムフィールドを使用して動的に変更できます。
